# LivePaper v1.0.1 Release Notes

Release date: 2026-07-14

## What's New in v1.0.1

### 🎨 Visual & UI Overhaul
- **New App & Menu Icons**: Redesigned main app and menu bar icons featuring custom vector-layered screens.
- **Glassmorphic Control Center**: Complete UI redesign with translucent materials, custom system metrics grid, pulsing status indicators, and a unified video library layout.
- **Custom Titlebar**: Integrated window traffic light controls directly into a transparent titlebar for a modern macOS aesthetic.

### 🛠️ Sleep & Wake Resiliency (Fixes)
- **Desktop Wallpaper Recovery**: Fixed wallpaper disappearance/reset after macOS sleep/wake cycles by elevating window level and adding workspace power observers.
- **Screensaver Sandbox & Recovery**: Resolved black screens and state corruptions after wake cycles caused by sandbox container paths and AVPlayer state desynchronization.
