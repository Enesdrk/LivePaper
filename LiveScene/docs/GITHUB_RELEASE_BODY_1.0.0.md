# Livepaper v1.0.0

Initial public release of Livepaper.

## Download
- `Livepaper-Local-1.0.0.dmg`

## Includes
- Live video wallpaper
- Live screen saver
- Separate media assignment for wallpaper and screen saver
- Menu bar Control Center
- Local-only config/status storage
- Privacy mode enabled by default

## Supported macOS
- macOS 13+

## Install
1. Download `Livepaper-Local-1.0.0.dmg`
2. Open DMG
3. Run `Livepaper-Local-1.0.0.pkg`
4. Open Livepaper from Applications
5. Select Livepaper in System Settings > Screen Saver

## Notes
- This release is local signed (ad-hoc), not notarized.
- If upgrading from older builds, uninstall first for clean setup.
